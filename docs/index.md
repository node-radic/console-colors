# @radic/console-colors

### Use
- Install trough `npm install @radic/console-colors`

#### Browser
No support

#### Node
ES5: 
`var cc = require('@radic/console-colors')`
ES6 / Typescript:
```typescript
import * as cc from '@radic/console-colors';
// or 
import {parser, Parser, supports} from '@radic/console-colors';
```
